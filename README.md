# 我的日记本

一个简洁美观的Vue 3日记本应用，以日历形式展示和管理日记。

## 功能特点

- 📅 **日历视图**：直观的月历展示，一目了然查看每月日记
- ✍️ **便捷记录**：点击任意日期即可写日记，无需跳转页面
- 📝 **多行支持**：支持多行文本，保留换行格式
- 💾 **本地存储**：使用浏览器本地存储，保护隐私
- 📱 **响应式设计**：适配手机、平板和桌面设备

## 技术栈

- Vue 3 - 渐进式JavaScript框架
- Vue Router - 单页应用路由
- Vite - 快速构建工具
- CSS3 - 现代样式和动画

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **查看日记**：点击日历上有内容的日期
2. **写日记**：点击空白日期，直接在弹窗中书写
3. **编辑日记**：在查看日记界面点击"编辑"按钮
4. **删除日记**：在查看日记界面点击"删除"按钮
5. **切换月份**：使用顶部的左右箭头切换月份
