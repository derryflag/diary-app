# 我的日记本

一个简洁美观的Vue 3应用，包含日记管理和相册功能，支持云存储和本地部署。

## 功能特点

### 📅 日记功能
- 📅 **日历视图**：直观的月历展示，一目了然查看每月日记
- ✍️ **便捷记录**：点击任意日期即可写日记，无需跳转页面
- 📝 **多行支持**：支持多行文本，保留换行格式
- 💾 **本地存储**：使用浏览器本地存储，保护隐私

### 📸 相册功能
- 🖼️ **照片管理**：支持上传、浏览、预览和删除照片
- 🎥 **视频管理**：支持视频上传、自动压缩和在线播放
- ☁️ **云存储**：集成阿里云OSS，支持云端存储和访问
- 📱 **响应式网格**：可调节每行显示数量，适配各种屏幕
- 🏷️ **日期分组**：自动按拍摄日期分组展示
- 🔄 **智能压缩**：视频自动压缩至720p，节省存储空间
- 💾 **双重存储**：云端和本地同时存储，提高访问速度

## 技术栈

- Vue 3 - 渐进式JavaScript框架
- Vue Router - 单页应用路由
- Vite - 快速构建工具
- Express - Node.js Web框架
- Sharp - 图像处理库
- FFmpeg - 音视频处理工具
- 阿里云OSS - 对象存储服务
- CSS3 - 现代样式和动画

## 快速开始

### 安装依赖

```bash
npm install
```

### 配置环境变量

```bash
cp server/.env.example server/.env
```

编辑 `server/.env` 文件，填入阿里云OSS配置信息。

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 启动生产服务器

```bash
npm start
```

## 使用说明

### 日记部分
1. **查看日记**：点击日历上有内容的日期
2. **写日记**：点击空白日期，直接在弹窗中书写
3. **编辑日记**：在查看日记界面点击"编辑"按钮
4. **删除日记**：在查看日记界面点击"删除"按钮
5. **切换月份**：使用顶部的左右箭头切换月份

### 相册部分
1. **上传照片/视频**：点击右下角"上传"按钮选择文件
2. **浏览相册**：按日期分组的网格布局展示
3. **预览照片**：点击缩略图全屏查看
4. **播放视频**：点击视频缩略图后点击播放按钮
5. **切换视图**：右上角可调节每行显示数量（3/4/5个）
6. **删除文件**：长按缩略图或在预览模式下点击删除按钮
7. **下载文件**：在预览模式下点击下载按钮
